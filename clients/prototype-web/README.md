# Prototype Web Client (Action Sequences + Graphics)

Minimal playable visual prototype for game-feel iteration.

## Run
```bash
cd clients/prototype-web
python3 -m http.server 8080
```
Open `http://localhost:8080`.

## Included
- Animated crime and raid action sequences (Canvas)
- Resource HUD updates (cash/energy/influence)
- Action feed with success/failure events

This is a temporary graphics/client-feel prototype while Unity client production work continues.
